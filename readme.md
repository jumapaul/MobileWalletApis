# MobileWalletApis

## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/jumapaul/MobileWalletApis
   cd MobileWalletApis
   ```
2. **Variables addition**

- Add the necessary environment variables to properties file, and run.
